# 🌦️ AI Weather Tool using MCP & ADK

## 📌 Overview
This project is a simple AI weather tool that fetches real-time weather data using an API.

## 🎯 Features
- Get the weather by city
- API integration
- Error handling

## 📂 Structure
ai-agent-mcp-weather/
│── tools/
│   └── weather.py
│── main.py
│── requirements.txt
│── README.md

## ▶️ Run
pip install -r requirements.txt  
python main.py

## 🧪 Example
Enter city: Delhi  
Weather in Delhi: clear sky, Temperature: 28°C
